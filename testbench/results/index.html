VICE Testbench Results

From this page all the various results gathered by the VICE testsuite are linked. The linked pages have been created like this: 
  $EMUDIR="/home/foo/bar/z64k/" ./testbench.sh z64k
  $EMUDIR="/home/foo/bar/z64k/" ./testbench.sh z64kc128c64
  $EMUDIR="/home/foo/bar/hoxs64/" ./testbench.sh hoxs64 --pal --cianew --8565early
  $EMUDIR="/home/foo/bar/micro64/" ./testbench.sh micro64 --pal
  $EMUDIR="/home/foo/bar/emu64/" ./testbench.sh emu64
  $EMUDIR="/home/foo/bar/yace/" ./testbench.sh yace
  $./testbench.sh cham20 --8k
  $EMUDIR="/home/foo/bar/z64k/" ./testbench.sh z64kvic20
  $EMUDIR="/home/foo/bar/z64k/" ./testbench.sh z64kc128
and 
  $./testbench.sh vsid / x64 / x64sc / x128 / xplus4 / xpet / x64dtv / xscpu64 / xcbm2 / xcbm5x0
then 
  $./testresults.sh sid / c64 / vic20 / c128 / plus4 / pet / dtv / scpu / cbm2 / cbm5x0 --html
